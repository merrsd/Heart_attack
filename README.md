## Общая информация и цель проекта
Требуется разработать модель машинного обучения, а также подготовить библиотеку и
интерфейс к ней для предсказания на тестовой выборке. В начале проекта доступен обучающий датасет и тестовый датасет, не содержащий правильных ответов.
Метрика на тестовом датасете будет измерена в ходе проверки работы. 

### Описание данных:
Описание исходных данных:

*   id  - id
*   Антропометрические параметры:  
          -  'Age' возраст  
          -   'BMI' индекс массы тела  
          -   'Income' доход  
*   Привычки:  
          -   'Smoking' курение 
          -   'Alcohol Consumption' употребление алкоголя  
          -   'Exercise Hours Per Week', 'Physical Activity Days Per Week' физич нагрузка часов и дней в неделю  
          -   'Sedentary Hours Per Day' часы без активности в день  
          -   'Sleep Hours Per Day' продолжительность сна  
          -   'Diet' питание, 'Stress Level' уровень стресса  
*   Давление:  
          -   'Heart rate' пульс  
          -   'Systolic blood pressure' систолическое давление  
          -   'Diastolic blood pressure' диастолическое давление  
*   Наличие хронических заболеваний:  
          -  'Previous Heart Problems' предыдущие проблемы с сердцем  
          -  'Diabetes' диабет  
          -  'Family History' наследственные заболевания  
          -  'Medication Use' принимаемые лекарства  
*   Биохимия крови:  
          -  'Cholesterol' холестерин  
          -  'Blood sugar' уровень сахара в крови  
          -  'Triglycerides' жиры в крови  
          -  'CK-MB' фермент повреждения сердечной мышцы  
          -  'Troponin' маркер инфаркта  
*   Таргет:  
           - 'Heart Attack Risk' высокий или низкий риск поражения сердца  

## Задачи проекта:
1. Исследование датасета.
2. Предобработка данных. 
3. Обучение модели
4. Подготовка предсказания на тестовой выборке.
5. Подготовка скриптов и библиотеки для обработки данных и предсказания на
тестовой выборке.
6. Написание инструмента для тестирования
7. Оформление документации

## Результат проекта:
1. Проведено исследование и предобработка данных, добавление новых признаков в датасет
2. Для оптимизации использован Pipeline.
3. Использован ООП - обработка признаков вынесена в отдельный модуль.
4. Обучены модели. Лучше всего показала себя CatBoostClassifier.
5. Важно было увеличить recall, так как это задача связана с сердечными рисками. 
6. Реализован API с помощью FastAPI.
